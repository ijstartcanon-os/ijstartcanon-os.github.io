How to Print on Canon ij Printer

  1. Follow the procedure given here to print any file on your computer. Check if the Canon printer is turned on. Extend the input tray and load plain papers into it. Align the plain paper on a flat surface before you place it.
  2. Next, launch the Printer Driver’s Setup screen on your system. Choose the Commonly Used Settings options from the Quick Setup tab.
  3. Settings such as Additional Features, Media type, and Paper Size can be modified. Set the Print Quality as High, Standard, or Draft based on your needs.
  4. Click the OK button after changing the settings. Search for the document you want to print. Open it on your system.
  5. From the File tab, select the Print option. Make any changes in the Print Preview dialog box if needed. Tap Print.

CANON IJ Wireless Setup

  1. Long-press the ON button to turn On the Canon printer. Tap the Settings button on the Canon printer’s touch panel and then press the arrow key to go to the Device Settings window.
  2. Press OK and tap the arrow buttons until you switch to the Wireless LAN setup window. Select the OK button, and the Canon printer will start searching for wireless networks.
  3. Press the name of your Wireless Fidelity network, choose the OK button, and then type the password in the Enter Passphrase field.
  4. Press the OK button and wait until the Connected pop-up window displays on the touch panel. Add the Canon printer to your personal computer.
